# Customer Sentiment Monitoring Skill

## Objective
Track customer complaints, feature requests, and sentiment across public platforms. Identify patterns, detect at-risk customers, and provide actionable recommendations.

## Determine Run Type
1. Frequent signal gathering - see file `signal-gathering.md`
2. Daily Signal Summary Report - see file `report-creation.md`
